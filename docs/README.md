# Zoo Management System

This project is designed to manage different types of animals in a zoo. It provides the necessary classes and methods to operate the zoo effectively.

## Class Diagram
![Zoo Management System Class Diagram](ZooManagementSystemClassDiagram.png)

## Classes
- Animal
- Mammal
- Reptile
- Bird
- Zoo

### Usage
- Create instances of animal classes.
- Manage them using the Zoo class.
